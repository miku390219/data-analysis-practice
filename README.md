#  Data Analysis Practice

pandas・NumPy・scikit-learnを用いたデータ分析・前処理・モデル評価の学習記録リポジトリです。

---

##  ノートブック一覧

| ファイル | 内容 |
|---|---|
| `1-dataframe_practice.ipynb` | pandasを用いたデータ処理の導入 |
| `2_data_wrangling.ipynb` | 欠損値処理・正規化・標準化等 |
| `3-eda.ipynb` | seabornライブラリを用いたEDA（探索的データ分析） |
| `4_model_development.ipynb` | 線形回帰分析のモデル作成 |
| `5_model_evaluation.ipynb` | モデル評価とRidge回帰・Grid Searchを用いたパラメータ改善 |
| `6_model_practice.ipynb` | データ処理からモデル評価までの一連の流れ |

---

##  使用技術

| カテゴリ | ライブラリ |
|---|---|
| データ操作 | pandas, NumPy |
| 可視化 | matplotlib, seaborn |
| 機械学習 | scikit-learn |
| 実行環境 | Google Colab, Jupyter Notebook |

---

##  学習内容

- **前処理**：欠損値処理・カテゴリ変数エンコーディング・スケーリング
- **EDA**：分布確認・相関分析・外れ値検出
- **回帰**：線形回帰・多項式回帰・Ridge/Lasso
- **評価**：MSE・R²
