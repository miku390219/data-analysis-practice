# Data Analysis Practice

pandas・NumPy・scikit-learnを用いたデータ分析・前処理・モデル評価の学習記録リポジトリです。


---

## ノートブック一覧

| ファイル | 内容 |
|---|---|
| `1_Data_Manipulation.ipynb` | pandasを用いたデータ操作・前処理の基礎 |
| `Data2_OldCar.ipynb` | 中古車データセットを用いた回帰分析 |
| `Data3_EDA.ipynb` | 探索的データ分析（EDA）・seabornによる可視化 |
| `Data4_KDE.ipynb` | カーネル密度推定（KDE）による分布の可視化 |
| `Data5_modelEvaluation.ipynb` | モデル評価指標（精度・F1・ROC曲線・Jaccard係数）の実装 |
| `Data6_modelPractice.ipynb` | 各種モデルの実装・比較実験 |
| `Data_ラップトップ価格.ipynb` | ラップトップ価格データを用いた予測モデル構築 |
| `Data_研究室データセットインポート.ipynb` | 研究室データのインポート・前処理パイプライン |

---

## 使用技術

| カテゴリ | ライブラリ |
|---|---|
| データ操作 | pandas, NumPy |
| 可視化 | matplotlib, seaborn |
| 機械学習 | scikit-learn |
| 実行環境 | Google Colab, Jupyter Notebook |

---

## 学習内容

- **前処理**：欠損値処理・カテゴリ変数エンコーディング・スケーリング
- **EDA**：分布確認・相関分析・外れ値検出
- **回帰**：線形回帰・多項式回帰・Ridge/Lasso
- **評価**：MSE・R²・F1スコア・ROC-AUC・Jaccard係数
